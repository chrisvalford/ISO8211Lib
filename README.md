# ISO8211Lib

Experiment to wrap the ISO8211 catalog reader c++ library
from F Warmerdam (2005) 
Currently only boilerplate code for simple c++ functions.
Adding the libray components one at a time.

The associated test app is https://github.com/chrisvalford/ISO8211LibTester
It's a multi platform SwiftUI implementation, with an effort to keep the structure clean.
